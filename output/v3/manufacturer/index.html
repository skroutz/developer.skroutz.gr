Manufacturers API

List manufacturers

GET /manufacturers

view example

Retrieve a single manufacturer

  GET /manufacturers/:id

GET /manufacturers/12907

view example
Status: 200
{
  "manufacturer": {
    "id": 12907,
    "name": "Rapala",
    "image_url": null
  }
}
