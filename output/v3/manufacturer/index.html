Manufacturers API

  • List manufacturers
  • Retrieve a single manufacturer
  • Retrieve a manufacturer’s categories
  • Retrieve a manufacturer’s SKUs

List manufacturers

  GET /manufacturers

GET /manufacturers

view example
Status: 200
{
  "manufacturers": [
    {
      "id": 1,
      "name": "EA - Electronic Arts",
      "image_url": null
    },
    {
      "id": 2,
      "name": "Sony",
      "image_url": null
    }
  ]
}

Retrieve a single manufacturer

  GET /manufacturers/:id

GET /manufacturers/12907

view example
Status: 200
{
  "manufacturer": {
    "id": 12907,
    "name": "Rapala",
    "image_url": null
  }
}

Retrieve a manufacturer’s categories

Order

Valid Parameters to perform ordering

Name Type Description
order_by String popularity (default) | name
order_dir String asc (default) | desc 
  GET /manufacturers/:id/categories
