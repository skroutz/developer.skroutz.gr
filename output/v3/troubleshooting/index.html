why am i not seeing all my results?

most api calls accessing a list of resources (e.g., categories, skus, e.t.c.) support pagination. If you’re making requests and receiving an incomplete set of results, you’re probably only seeing the first page. You’ll need to request the remaining pages in order to get more results.

It’s important to not try and guess the format of the pagination URL.
