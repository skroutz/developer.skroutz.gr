About

Categories are organized in a tree-like structure.
A product/sku belongs to a leaf category.

Retrieve a single category

  GET /categories/:id

GET /categories/1442

view example
Status: 200
{
  "category": {
    "id": 1442,
    "name": "Ξαπλώστρες Κήπου & Βεράντας",
    "children_count": 0,
    "image_url": "http://b.scdn.gr/images/categories/large/1442.jpg",
    "leaf": true
  }
}

Retrieve the parent of a category

  GET /categories/:id/parent

GET /categories/1442/parent

view example
Status: 200
{
  "category": {
    "id": 1434,
    "name": "Έπιπλα Κήπου & Βεράντας",
    "children_count": 10,
    "image_url": "http://b.scdn.gr/images/categories/large/1434.jpg",
    "leaf": false,
    "parent_id": 1052
  }
}
