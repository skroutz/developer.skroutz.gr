About

Categories are organized in a tree-like structure.
A product/sku belongs to a leaf category.

List all categories

  • verb: GET
  • path: /categories
  • params:
    • page: 1
    • per: 5
    • oauth_token: [FILTERED]

Retrieve a single category

  • verb: GET
  • path: /categories/1442
  • params:
    • oauth_token: [FILTERED]
Status: 200
{
  "category": {
    "id": 1442,
    "name": "Ξαπλώστρες Κήπου & Βεράντας",
    "children_count": 0,
    "image_url": "http://b.scdn.gr/images/categories/large/1442.jpg",
    "leaf": true
  }
}

If you request the skus of a non-leaf category you’ll get a 404 Not Found response.

    • per: 3
    • oauth_token: [FILTERED]
