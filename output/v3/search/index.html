Search

Parameters

q
Required string The keyphrase to search by

Case invalid The query was invalid. Eg. less than 2 characters

  • verb: GET
  • path: /search
  • params:
    • q: a
    • oauth_token: [FILTERED]
Status: 400
{
  "meta": {
    "result_type": "invalid"
  },
  "message": "Η αναζήτησή σου πρέπει να περιέχει τουλάχιστον 2 χαρακτήρες"
}

Case no_results Nothing seems to match the specified query

  • verb: GET
  • path: /search
  • params:
    • q: asdasdsagjgjgjweigrweji
    • oauth_token: [FILTERED]
Status: 200
{
  "meta": {
    "result_type": "no_results"
  }
}

Case manufacturer_match *Your query matched one of our manufacturers

exactly*

  • verb: GET
  • path: /search
  • params:
    • q: apple
    • oauth_token: [FILTERED]
Status: 200

    "id": 356,
    "name": "Apple",
    "image_url": null

  }
}

Case drop *Your query has no results but the proposals in the

response have*

  • verb: GET
  • path: /search
  • params:
    • q: wrong iphone
    • oauth_token: [FILTERED]
Status: 200
{
  "proposals": [
    "wrong",
    "iphone"
  ],
  "meta": {
    "result_type": "drop"
  }
}
