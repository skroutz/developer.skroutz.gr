The addOrder command creates a shopping cart object.

Data Parameters:
##### Example ```javascript sa('ecommerce', 'addOrder', { 'order_id': '123456', // Order ID. Required. 'revenue': '120.99', // Grand Total. 'shipping': '5.45', // Total Shipping Cost. 'tax': '10.50' // Total Tax. }); ``` #### addItem The `addItem` command adds a new item to the shopping cart object. ##### Data Parameters:
Value Type Required Description
order_id String Yes The Order ID that produced and stored at your e-shop to identify the transaction. This id is what links items to the orders to which they belong. (e.g. 123456)
revenue Currency String No The total revenue or grand total of the order (e.g. 120.99). This value may include shipping, tax costs, or other adjustments to total revenue that you want to include as part of your revenue calculations.
shipping Currency String No The total shipping cost of the order. (e.g. 5.45)
tax Currency String No The total tax of the order. (e.g. 10.50)
Value Type Required Description
order_id String Yes The Order ID that produced and stored at your e-shop to identify the transaction. This id is what links items to the orders to which they belong. (e.g. 123456)
product_id String Yes The id of the product as stored at your e-shop. (e.g. 987)
price Currency String Yes The individual, unit, final price for each item. (e.g. 10.50)
quantity String Yes The number of units purchased in the order. (e.g. 4)
