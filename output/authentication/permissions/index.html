Permissions

A representations of the permissions granted to your app by Skroutz and a User.

Code Type Description
public application All public resources*
favorites user User Favorites
notifications user User notifications

For more info about permission types check authentication flows.

Public Resources

Having obtained the public permission you can access the following resources:

  • Categories
  • Skus
  • Products
  • Shops
  • Manufacturers
  • Search
  • Filter Groups

User Resources

These permissions require that you have explicitly requested them using the scope param see here The user will have to consent to grant any of those permissions. You must not share these data to 3rd parties.

  • Favorites
  • Notifications
